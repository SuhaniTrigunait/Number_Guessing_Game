## Number Guessing Game (Java)

## Description

This is a simple console-based number guessing game written in Java.
The program generates a random number between 1 and 100, and the user tries to guess it.

## Features

* Random number generation
* User input handling
* Attempt limit (unlimited)
* Input validation (only numbers between 1 and 100 allowed)
* Feedback for high/low guesses

## How to Run

1. Make sure Java is installed
2. Compile the program:

   ```
   javac NumberGuessingGame.java
   ```
3. Run the program:

   ```
   java NumberGuessingGame
   ```

## Concepts Used

* Loops (`while`)
* Conditional statements (`if-else`)
* Random number generation
* User input using `Scanner`
* Basic defensive programming
